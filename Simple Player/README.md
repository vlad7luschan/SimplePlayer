# Simple Player 🎵🎥

**Simple Player** — це мінімалістичний, сучасний та швидкий медіаплеєр, розроблений на базі фреймворку **.NET MAUI (WinUI 3)** для Windows. Додаток має унікальний кастомний дизайн інтерфейсу користувача, оптимізований для комфортного керування аудіо- та відеофайлами.

## ✨ Особливості (Features)

*   **Custom UI / Dark Theme:** Повністю унікальний темно-синій інтерфейс із кастомними елементами керування, кнопками та повзунками треку й гучності.
*   **Фірмовий брендинг:** Інтегровано оригінальну синю іконку додатка, яка відображається у Провіднику Windows, на Панелі завдань та в заголовку вікна (Title Bar).
*   **Автономність:** Додаток оптимізовано для збирання в єдиний незалежний файл або класичний інсталятор Windows.

# Simple Player 🎵🎥

**Simple Player** is a minimalist, modern, and lightweight media player built on top of the **.NET MAUI (WinUI 3)** framework for Windows. The application features a fully customized, unique user interface optimized for a clean and comfortable audio and video playback experience.

---

## ✨ Features

*   **Custom UI / Dark Theme:** A unique deep-blue custom user interface featuring stylized control buttons, track progress bars, and volume sliders.
*   **Custom Branding & Icons:** Seamless integration of a custom blue app icon that perfectly renders in Windows Explorer, the Taskbar, and the window Title Bar.
*   **Self-Contained Deployment:** Optimized configuration for single-file publishing or traditional Windows setup installation.
